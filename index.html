Pruebas msthompson

Pruebas msthompsen es un proyecto de sustentabilidad que facilita el diagnóstico de la calidad del agua en entornos naturales y de consumo. Utiliza un kit con puntero láser, switch, jeringa para muestras de agua, y soporte para una amplificación precisa de gotas de agua. La aplicación web analiza imágenes proyectadas para detectar microorganismos o bacterias en las muestras. Desarrollado con Laravel para el backend, React para la interfaz de usuario, y Tailwind CSS para el diseño.

Diseño eficiente y totalmente responsivo, adaptándose fluidamente a distintos dispositivos y tamaños de pantalla.
